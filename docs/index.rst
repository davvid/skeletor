======================
skeletor Documentation
======================

`skeletor <https://github.com/davvid/skeletor>`_ is a Python library for
reusable SQLAlchemy utilities.  It contains modules that can be considered
a minimalist "application skeleton", thus "skeletor".

Download & Install
==================

The easiest way to get skeletor is via PyPi_ with pip_::

    $ pip install -U skeletor

You can also download or :ref:`clone <contrib-clone>` the latest code and
install from source::

    $ python setup.py install

.. _PyPi: https://pypi.python.org/pypi/skeletor
.. _pip: https://pypi.python.org/pypi/pip
